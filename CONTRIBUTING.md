# Contributing to Mason Brick Generator Project

We welcome contributions to the Mason Brick Generator Project! By contributing, you help make this project better for everyone. Here are some guidelines to help you get started.

## How to Contribute

### Reporting Issues

If you find a bug or have a feature request, please create an issue on the [GitHub Issues](https://github.com/zoocityboy/mason_bundler/issues) page. Provide as much detail as possible to help us understand and address the issue.

### Fork the Repository

1. Fork the repository by clicking the "Fork" button on the GitHub page.
2. Clone your forked repository to your local machine:

    ```sh
    git clone https://github.com/<your-username>/mason_bundler.git
    cd mason_bundler
    ```

### Create a Branch

Create a new branch for your feature or bugfix:

```sh
git checkout -b my-feature-branch
